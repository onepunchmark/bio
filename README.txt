UPLOAD AUF GITHUB
1. ZIP-Datei entpacken.
2. Alle sechs Dateien direkt in dasselbe GitHub-Repository hochladen.
3. Die vorhandenen Dateien bei GitHub ersetzen.
4. "Commit changes" bestätigen.
5. Etwa 1–3 Minuten warten und die Seite neu laden.

WICHTIG
Die Dateien dürfen nicht in einem zusätzlichen Unterordner liegen.
Die Datenschutzerklärung ist auf eine statische GitHub-Pages-Seite ohne eigenes Tracking,
Kontaktformular, externe Schriftarten oder eingebettete Social-Media-Inhalte zugeschnitten.
Bei späteren Erweiterungen muss sie angepasst werden.
